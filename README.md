OpenClassrooms Projet 3 : Mettez votre logique à l'épreuve.
Il est possible de lancer l'application via Eclipse. 
Il est possible de lancer l'application en utilisant le fichier OC_Projet3_Jeux_RecherchePlusMoins_Et_Mastermind.jar :
 - Pour cela, décompressez le fichier dans un répertoire.
 - Lancer l'Invite de commandes depuis le répertoire où le fichier a été décompressé.
 - Dans l'invite de commandes, exécuter la commande java -Dlog4j.configurationFile=resources/log4j2.xml -jar OC_Projet3_Jeux_RecherchePlusMoins_Et_Mastermind.jar
 - Répondre à la question posée concernant l'activation du mode développeur (O pour oui/N pour non).
 - L'application se lance.
